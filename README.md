# 🛡️ cyber-project-05-incident-response-playbook
Incident response playbook for brute force attack (SOC simulation)


## Context

Incident response is a critical function in cybersecurity.

This project defines a structured response to a brute force attack.

---

## Objective

* Define response steps
* Standardize incident handling
* Simulate SOC procedures

---

## Scenario

Brute force attack detected on an admin account.

---

## Playbook Steps

### 1. Detection

* Identify repeated failed login attempts
* Confirm suspicious activity

---

### 2. Analysis

* Identify source IP
* Review logs
* Confirm attack pattern

---

### 3. Containment

* Block the attacker IP
* Lock affected account

---

### 4. Eradication

* Remove unauthorized access
* Reset credentials

---

### 5. Recovery

* Restore normal operations
* Monitor for further activity

---

### 6. Lessons Learned

* Improve monitoring rules
* Enforce MFA

---

## Key Takeaways

* Incident response workflow
* SOC procedures
* Cyber defense strategy
